#  Research Tracker System

A backend application built using **Spring Boot** to manage research projects, users, and authentication.  
It provides JWT-based authentication, user registration, and project management APIs.

---

##  Technologies Used

- **Java 17+**
- **Spring Boot 3.x**
- **Spring Security (JWT Authentication)**
- **Spring Data JPA**
- **Hibernate**
- **MySQL / PostgreSQL**
- **Lombok**
- **Maven**
